Feuille 4: Documents en HTML

Dans cette feuille, nous allons créer des documents en HTML. On rappelle que vous pouvez trouver les principales balises HTML dans la première partie du Cours n°2.

Nous ne nous occuperons pas de CSS dans cette feuille de TD. Ces dernières feront l'objet de la prochaine feuille.

Par ailleurs, pour ceux qui veulent aller plus loin, de nombreuses ressources sont aussi disponibles sur le Web.


Quelques rappels

Document HTML5 minimal

On trouvera ci-dessous un document HTML5 minimal. Vous pourrez vous en servir pour démarrer vos exercices:

<!DOCTYPE html>
<html>
  <head>
    <title>Titre du doument HTML5</title>
    <meta charset="utf-8">
  </head>
  <body>
    <p>...</p>
  </body>
</html>

Tables

Des exemples de tables sont donnés dans le Cours. Les principales balises que l'on peut utiliser pour les tables sont données ci-dessous:

Elément Nom Description
table table Déclare une table. Cette table peut contenir optionnellement une légende (caption). On définit ensuite les lignes avec l’élément tr et,dans chaque ligne, les cases avec l’élément td ou th .
tr table row Déclare une ligne d’un tableau de données. Elle permet de regrouper les cellules d’une même ligne.
th table header cell La balise <th> est une cellule d’en-tête d’un tableau de données.
td table data cell La balise <td> est une cellule d’un tableau de données. Elle peut contenir plusieurs éléments. Les attributs colspan et rowspan permettent de définir des cellules qui débordent sur les colonnes ou lignes suivantes.
caption caption Contient la légende d’une table.

Il existe d'autres balises utilisables dans les tables (column, colgroup, thead, tbody et tfoot). Elles ne seront pas utilisées dans ce TD.

Entités

Certains caractères ne sont pas accessibles directement dans un document HTML car ils ont une signification particulière (par exemple le caractère < qui dénote le début d'une balise). Pour afficher ces caractères, on utilise alors des entités:

Caractère Entité
espace insécable (non breaking space) &nbsp;
'<' (less than)` &lt;
'>' (greater than) &gt;
'&' (ampersand) &amp;
'"' (quote) &quote

Exercices

Un document erroné

Prenez le document villes.html. Ce fichier n'est pas valide. Copiez le fichier chez vous et rendez le valide. Un fois que c'est fait, vous pouvez vérifier que votre fichier est bien valide sur le validateur du W3C.

Remarques:

  • Vous avez peut-être remarqué que la page erronée s’affiche «assez» correctement. Toutefois, seule la validité de vos pages vous assure que l’affichage est «réellement» comme vous le voulez et surtout qu’il sera conforme à vos attentes sur tous les navigateurs.

  • Contrairement à ce qui a été dit plus haut, on utilise ici une balise <style> (et donc on utilise les CSS) dans le fichier donné. Les informations données dans cette balise permettent de mettre des bordure autour des cellules de la table (par défaut, les tables HTML n'ont pas de bordure). Vous n'avez pas ici à modifier/comprendre ce qui se trouve entre les balises <style> et </style>.
